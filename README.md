# Next.js Course Modules

This repository contains the code and exercises for Module 4 of the Coding course by SMIT, focusing on Next.js.

## Class Curriculum

### [Class 1](./class-1)
**Foundations of Routing**
- Route Groups: `(auth)`, `(dashboard)`, `(home)`
- Nested Layouts
- Basic page structure

### [Class 2](./class-2)
**Components & Dynamic Routing**
- Reusable Components (`Navbar`, `Footer`)
- Dynamic segments (`[productsId]`)
- Basic API introduction

### [Class 3](./class-3)

- Project initialization and structure practice.

## Advanced Topics (Newly Added)

### [Class 9](./class-9)
**Data Fetching Strategies**
- **Server Components:** Fetching data directly on the server (`async/await`).
- **Client Components:** Fetching data on the browser (`useEffect`).
- Comparison of both approaches.

### [Class 10](./class-10)
**Middleware & Authentication**
- Creating `middleware.ts`.
- Protecting routes (redirecting unauthenticated users).
- Managing cookies for session states in Next.js.
- `login` vs `dashboard` access control.

### [Class 11](./class-11)
**Server Actions (Mutations)**
- `"use server"` directives.
- Submitting forms without API routes.
- `revalidatePath` for instant UI updates.
- Implementing simple CRUD (Create, Read, Delete) for a Todo list.
